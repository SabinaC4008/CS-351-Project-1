Midnight Wings

Creator: Jana Serhii
Item Descripton:
These earrings were created for Jana Serhii’s Far Away Land jewelry line. The line was created for the 2023 Celestial Jewel Showing which was themed Mythology. This line came in 3nd during the CJ 2023 competition that accompanies the 2023 Celestial Jewel Showing. The line was created in collaboration with designer Nokomis Serhii’s Draconic Fae fashion line.

These earrings were inspired by Faeries and Fairies of European folklore. The specific inspiration was the western animation series Winx Club.
